# Readme  
这里包含了该项目的注意事项  

## 1.分支管理
分支管理按照以下结构进行：

main  
└── dev  
├── module/resource  
│ ├── feature/resource-upload  
│ ├── feature/resource-download  
│  
├── module/auth  
│ ├── feature/auth-login  
│ ├── feature/auth-jwt  

- `main` 是主分支  
- `dev` 是开发分支  
- `module/resource` 是模块分支  
- `feature/resource-upload` 是模块开发的功能分支  

> **注意**：创建分支时请确认上一级的分支。
